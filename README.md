pyqpbo
======

QPBO interface and alpha expansion for Python.

Running ``make`` downloads QPBO by Vladimir Kolmogorov
from http://pub.ist.ac.at/~vnk/software/QPBO-v1.3.src.tar.gz
and builds the Python package.
